# SPMB SMP Negeri 3 Kawangkoan

Aplikasi Pendaftaran Murid Baru Online untuk SMP Negeri 3 Kawangkoan Tahun Pelajaran 2026/2027.

## Fitur

- ✅ Formulir pendaftaran online
- ✅ Validasi data real-time
- ✅ Generate nomor registrasi otomatis
- ✅ Download formulir dalam format PDF
- ✅ Video tutorial pendaftaran
- ✅ Countdown batas akhir pengumpulan
- ✅ Responsive design (bisa di HP)

## Teknologi

- HTML5, CSS3, JavaScript
- sheetDB.io untuk backend database
- jsPDF untuk generate PDF
- YouTube API untuk video tutorial
- Netlify untuk hosting

## Cara Penggunaan

1. Buka website: [https://spmb-smpn3-kawangkoan.netlify.app](https://spmb-smpn3-kawangkoan.netlify.app)
2. Baca informasi dan persyaratan
3. Klik "Daftar Sekarang"
4. Isi formulir dengan lengkap dan benar
5. Klik submit
6. Download formulir PDF
7. Bawa formulir cetak + kelengkapan ke sekolah
